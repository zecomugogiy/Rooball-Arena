# App Store Notes

## Review Notes

Rooball Arena's native tools work without an account, login, subscription, or special permission. The app may perform a launch check for optional web content.

Main flows to test:

1. Open Match.
2. Edit both team names.
3. Start and pause the period clock.
4. Add goals and shots for both teams.
5. Save the match.
6. Open Learn and switch between English, French, and Spanish.
7. Complete the quiz and delete local match history.

## Age Rating Inputs

- Cartoon or fantasy violence: None
- Realistic violence: None
- Profanity: None
- Mature themes: None
- User-generated content: None
- Web access: Optional launch-controlled web content
- Gambling: None

## Export Compliance

The app does not use encryption beyond standard Apple platform behavior.
