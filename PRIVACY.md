# Privacy Policy

Rooball Arena keeps native match notes, settings, scores, and language preferences on device.

On launch, the app may send a server check to `goldapp.ink` using the app bundle identifier and configured app token to determine whether optional web content should open. The app does not ask for names, email, contacts, location, camera, or microphone access.

The app stores scoreboard state and saved match history locally on the user's device with UserDefaults. This information is not sent to any server and is not available to the developer.

The native app does not use ads, location services, camera, microphone, contacts, or account login.

Users can delete saved match history from the Learn tab.
